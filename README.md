# 📊 Data Analytics Project

## Overview

This project showcases an end-to-end data analytics workflow, transforming raw data into meaningful business insights. The project covers data loading, exploratory data analysis (EDA), data cleaning, SQL analysis, dashboard development in Power BI, business reporting, and presentation of findings.

The objective is to demonstrate practical data analytics skills using industry-standard tools and techniques.

---

## Dataset

The dataset contains business transactional data used to analyze sales performance, customer behavior, product performance, and profitability.

## Project Workflow

### 1. Data Loading
- Imported the dataset into Python.
- Inspected the data structure and data types.

### 2. Exploratory Data Analysis (EDA)
- Explored the dataset.
- Generated summary statistics.
- Identified missing values and duplicates.
- Analyzed distributions and relationships between variables.

### 3. Data Cleaning
- Removed duplicate records.
- Handled missing values.
- Corrected data types.
- Standardized column names.
- Prepared the dataset for analysis.

### 4. SQL Analysis
- Imported the cleaned dataset into Microsoft SQL Server.
- Wrote SQL queries to answer business questions.
- Used aggregations, joins, CTEs, and window functions to generate insights.

### 5. Power BI Dashboard
- Built an interactive dashboard.
- Created KPI cards and visualizations.
- Added filters and slicers for dynamic analysis.
- Presented key business metrics and trends.

### 6. Business Report
- Summarized key findings.
- Highlighted business insights.
- Provided data-driven recommendations.

### 7. Presentation
- Created a professional presentation in Gamma to communicate the project methodology, findings, and recommendations.

## How to Run

1. Clone this repository.
2. Install the required Python libraries:

```bash
pip install -r requirements.txt
```

3. Run the Python notebook or script.
4. Import the cleaned dataset into Microsoft SQL Server.
5. Execute the SQL queries.
6. Open the Power BI dashboard (`.pbix`).
7. Review the business report and presentation.
